### Compact encoding for Cyrillic with different levels of compression

Swap Cyrillic characters to ASCII lookalikes for Meshtastic. Works as a PWA on Android and iOS, can be installed from a browser for offline use.

Компактное кодирование кириллицы для Meshtastic. Замена русских символов на английские с несколькими уровнями сжатия. Работает оффлайн на Android и iOS.
